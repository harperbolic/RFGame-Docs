# Diretrizes e Boas Práticas

## Boas práticas da Godot Engine

As diretrizes de código do projeto serão, por consistência, as mesmas recomendadas pela Godot Engine.

Para diretrizes de código em GDScript, utilize o [GDScript style Guide](https://docs.godotengine.org/en/stable/tutorials/scripting/gdscript/gdscript_styleguide.html)

Para boas práticas de organização de arquivos e outras coisas in-engine, utilize o [Manual de Boas Práticas da Godot Engine](https://docs.godotengine.org/en/stable/tutorials/best_practices/index.html).
