# Fundamentos

Para começar, vamos aprender e nos familiarizar com o GDScript, a linguagem de programação da Godot Engine. Uma linguagem de scripting muito similar com Python e fácil de aprender.

A principal recomendação para aprender GDScript é um jogo feito em Godot que utiliza a linguagem para resolver fases. Apenas isso é o suficiente para entender a lógica e sintaxe básica e posteriormente, continuar o aprendizado através da prática.

- Jogue: [https://gdquest.itch.io/learn-godot-gdscript](https://gdquest.itch.io/learn-godot-gdscript)
