# Fundamentos

Para começar, vamos aprender e nos familiarizar com o GDScript, a linguagem de programação da Godot Engine. Uma linguagem de scripting muito parecida com Python e fácil de aprender.

Normalmente, a melhor forma de aprender sobre algo é a documentação oficial. A Godot Engine possui uma excelente página de 'Getting Started':

- Leia: [Godot Engine Docs - Getting Started](https://docs.godotengine.org/en/stable/getting_started/introduction/index.html)

Porém, caso você prefira uma abordagem mais interativa, há um excelente jogo que ensina o GDScript.

- Jogue: [https://gdquest.itch.io/learn-godot-gdscript](https://gdquest.itch.io/learn-godot-gdscript)
