# Manual do Programador

Boas-vindas ao Manual do Programador. Essa seção tem como objetivo nivelar o conhecimento de todos programadores da equipe e servir como material de consulta durante o desenvolvimento.

Os tópicos abordados são:

- Materiais de Estudo
    - Com ênfase na Godot Engine e programação em GDScript.
- Diretrizes de Código
    - Padronizações ao escrever certas coisas para melhorar o entendimento do código e a consistência.
- Git e Colaboração
    - Controle de versão para permitir que toda quipe colabore de forma eficiente.
